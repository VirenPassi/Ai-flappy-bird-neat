# AI Flappy Bird (Neuroevolution)

An implementation study of the NEAT (NeuroEvolution of Augmenting Topologies) algorithm to train autonomous agents capable of learning optimal flight paths through dynamic obstacles over successive generations.

## Tech Stack
* Python
* Pygame
* neat-python
* numpy
* graphviz
* matplotlib

## Acknowledgements
This project was developed as a foundational learning exercise in neuroevolution, utilizing the core structural architecture from Tech With Tim's NEAT tutorial series.
